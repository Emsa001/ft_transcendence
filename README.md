## 🧪 Tech Stack
- **TypeScript** — Because JavaScript wasn’t painful enough.
- **WebSockets** — For that sweet real-time lag.
- **PostgreSQL** — No. Just kidding. **SQLite** — Because nothing says “production ready” like a file you can delete by accident.
- **Sequelize** — So we can pretend SQL is JavaScript and still get errors.
- **Docker** — Not because we hate ourselves, but because it makes us feel like elite hackers while just running `docker compose up`.
- **ft_react** — Our custom React framework: 2x slower, 3x buggier, 10x more personal.
- **42 Tears™** — Proprietary ingredient.
